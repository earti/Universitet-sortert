Def:
Den tomme relasjonen (eng. empty relation) fra $S$ til $T$, som ikke [[Relasjon|relaterer]] noen elementer til hverandre: $\emptyset$. Den tomme relasjonen pÃ¥ [[Mengde|mengden]]  $\{1,2,3\}$ er $\emptyset$.

|     |     |     |     |
| :---: | :---: | :---: | :---: |
|     | $1$ | $2$ | $3$ |
| $1$ | $0$ | $0$ | $0$ |
| $2$ | $0$ | $0$ | $0$ |
| $3$ | $0$ | $0$ | $0$ | 
> [!quote] %%Referanser%%
Antonsen, Roger. *Logiske metoder*. Universitetsforlaget, 2014.
